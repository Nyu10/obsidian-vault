### Navigation and Action Hotkeys

These are shortcuts using modifier keys to perform actions:

- **Open Note in New Tab or Create Linked Note:** `Command` + `Click` (Mac) / `Control` + `Click` (Windows) [[02:40](http://www.youtube.com/watch?v=z4AbijUCoKU&t=160)] and [[04:27](http://www.youtube.com/watch?v=z4AbijUCoKU&t=267)]
    
- **Navigate Back or Forward:** `Command` + `Option` + `Left` or `Right` keys (Mac) / `Control` + `Alt` + `Left` or `Right` keys (Windows) [[09:24](http://www.youtube.com/watch?v=z4AbijUCoKU&t=564)]
    
- **Find Text on Page:** `Command` + `F` (Mac) / `Control` + `F` (Windows) [[09:30](http://www.youtube.com/watch?v=z4AbijUCoKU&t=570)]
    
- **Open New Tab:** `Command` + `T` (Mac) / `Control` + `T` (Windows) [[09:33](http://www.youtube.com/watch?v=z4AbijUCoKU&t=573)]
    
- **Close Tab:** `Command` + `W` (Mac) / `Control` + `W` (Windows) [[09:35](http://www.youtube.com/watch?v=z4AbijUCoKU&t=575)]
    
- **Open the Command Palette:** `Command` + `P` (Mac) / `Control` + `P` (Windows) [[09:50](http://www.youtube.com/watch?v=z4AbijUCoKU&t=590)]. This is also used to access commands like **Insert Table** [[12:44](http://www.youtube.com/watch?v=z4AbijUCoKU&t=764)].
    
- **Open the Quick Switcher:** `Command` + `O` (Mac) / `Control` + `O` (Windows) [[10:04](http://www.youtube.com/watch?v=z4AbijUCoKU&t=604)]
    
- **Add a New Property:** `Command` + `;` (Mac) / `Control` + `;` (Windows) [[13:09](http://www.youtube.com/watch?v=z4AbijUCoKU&t=789)]
    

### Text Formatting and Markdown Syntax Shortcuts

These are shortcuts based on Markdown syntax, which allows you to format text without using a mouse or menu:

- **Create and Link a Note:** Hit the **left bracket twice** (`[[`) to start linking a note [[04:00](http://www.youtube.com/watch?v=z4AbijUCoKU&t=240)], and hit the **right bracket twice** (`]]`) to finish [[04:10](http://www.youtube.com/watch?v=z4AbijUCoKU&t=250)].
    
- **Bold:** `Command` + `B` (Mac) / `Control` + `B` (Windows) [[08:38](http://www.youtube.com/watch?v=z4AbijUCoKU&t=518)]
    
- **Italicize:** `Command` + `I` (Mac) / `Control` + `I` (Windows) [[08:38](http://www.youtube.com/watch?v=z4AbijUCoKU&t=518)]
    
- **Headings:** Use the pound symbol (`#`) at the beginning of a line for different heading sizes [[08:43](http://www.youtube.com/watch?v=z4AbijUCoKU&t=523)].
    
- **Strike Through:** Surround text with two tildas (`~~`) [[08:50](http://www.youtube.com/watch?v=z4AbijUCoKU&t=530)].
    
- **Highlight:** Surround text with two equal signs (`==`) [[08:54](http://www.youtube.com/watch?v=z4AbijUCoKU&t=534)].
    
- **Quote Block:** Use a greater than sign (`>`) [[08:54](http://www.youtube.com/watch?v=z4AbijUCoKU&t=534)].
    
- **Bullet List:** Use a dash (`-`) [[08:58](http://www.youtube.com/watch?v=z4AbijUCoKU&t=538)].
    
- **Numbered Lists:** Use numbers (e.g., `1.`) [[08:59](http://www.youtube.com/watch?v=z4AbijUCoKU&t=539)].
    
- **Checklist:** Use a dash plus brackets (`- [ ]`) [[09:02](http://www.youtube.com/watch?v=z4AbijUCoKU&t=542)].
    
- **Divider:** Use three dashes (`---`) on an empty line [[09:07](http://www.youtube.com/watch?v=z4AbijUCoKU&t=547)].
    
- **Inline Code:** Use one back tick (`` ` ``) [[09:10](http://www.youtube.com/watch?v=z4AbijUCoKU&t=550)].
    
- **Code Block:** Use three back ticks (` ``` `) [09:11].
    
- **External Link:** Put the link text in brackets and the URL in parenthesis (`[text](url)`) [09:19].
    
- **Embed Note:** Add an exclamation point (`!`) before a note link (`![[Note Name]]`) to embed the entire note [09:38].
    
- **Display Image:** Add an exclamation point (`!`) before an external link to display an image [09:44].
